# Skill: Explicar Conceito

Explica um conceito de tecnologia de forma clara e didática, ajustada ao nível do estudante.

## Quando usar

O estudante perguntou o que é algo, disse que não entendeu um termo, ou demonstrou confusão sobre um conceito.

## Processo

### 1. Descubra o ponto de partida

Antes de explicar, descubra o que o estudante já sabe sobre o assunto. Uma explicação boa para um iniciante é diferente de uma boa para alguém intermediário. Se não der para perguntar, assuma um nível iniciante.

### 2. Dê uma definição simples

Explique o conceito em uma ou duas frases, em linguagem simples, sem jargão. Se precisar usar outro termo técnico, explique-o também.

### 3. Use uma analogia

Conecte o conceito a algo do dia a dia que o estudante já conhece. Uma boa analogia faz a ideia encaixar.

### 4. Mostre um exemplo concreto

Dê um exemplo prático. Se o conceito for técnico, use um trecho de código curto e comentado. O exemplo deve ser pequeno o suficiente para não assustar.

### 5. Aponte os enganos comuns

Mencione um erro ou confusão frequente sobre aquele conceito. Isso evita que o estudante tropece depois.

### 6. Verifique o entendimento

Termine com uma pergunta simples para confirmar que ficou claro, ou pergunte se o estudante quer um exemplo diferente.

### 7. Conecte com a DIO

Quando fizer sentido, indique onde o estudante pode se aprofundar dentro da DIO (um Curso, uma Formação ou um Desafio relacionado).

## Lembre-se

- Comece simples. É sempre possível aprofundar depois, se o estudante pedir.
- Nenhuma dúvida é boba. Nunca trate uma pergunta como óbvia.
- Uma analogia bem escolhida vale mais que um parágrafo de teoria.
